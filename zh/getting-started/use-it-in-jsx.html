在JSX中使用增强React开发体验

NornJ语法通过配套的babel插件可直接在JSX中编写，例如：

const test = props => (
  <if condition={props.isTest}>
    <i>success</i>
    <else>
      <i>fail</i>
    </else>
  </if>
);

下面的是一个在线可运行实例：

  • 在线Playground(codesandbox)

安装

npm install babel-plugin-nornj-in-jsx@next  #or yarn add babel-plugin-nornj-in-jsx@next

然后配置.babelrc:

{
  "plugins": [
    "nornj-in-jsx"
  ]
}

具体请见babel-plugin-nornj-in-jsx。

与各种React已有生态结合

NornJ可直接支持所有React现有生态，包括Redux、React-Router、Mobx、Ant Design等等，它可以和任何已有的React生态共存。
