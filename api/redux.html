Redux

结合Redux

可以直接在NornJ模板中使用Redux提供的组件：

import { Provider } from 'react-redux';

nj`
  <${Provider} store=${store}>
    <${App} />
  </${Provider}>
`()

使用nornj-react包中的redux组件

在nornj-react包中提供一些默认封装的组件，可更方便地使用Redux。

这样引入即可：

import 'nornj-react/redux';
